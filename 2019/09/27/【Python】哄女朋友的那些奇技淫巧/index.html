下面这个是我们的二维码生成器，可以将文字，号码，网址存放在二维码中，只需要只用make方法去生成二维码，如何使用save方法保存二维码图片即可。很简单的两部搞定！下面我们来看一个栗子：

jieba

jieba，真的就是结巴的意思，主要是用来进行分词的。

jieba 可以**把一句话拆分成多个词。**

从一句话（一段话）中提取最重要的几个关键词。

最常用的功能应该就是这些吧，分词之后结合 TF-IDF，就可以做搜索工具和相关推荐了。
